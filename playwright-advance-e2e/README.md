# Playwright Advanced E2E Framework Skill

An enterprise-grade QA skill for AI coding agents — teaches your AI agent to build Playwright test frameworks using 8-layer architecture with Page Object Model and Module Pattern.

## Install

```bash
npx @qaskills/cli add playwright-advance-e2e
```

## What Your AI Agent Learns

- 8-layer architecture (Config → Pages → Modules → Utils → API → Fixtures → Reporting → CI/CD)
- Page Object Model with locators as arrow functions
- Module Pattern for business logic separation
- Custom Playwright fixtures for dependency injection
- API testing layer with typed responses
- Structured logging, data generators, wait helpers
- `test.step()` for clear HTML reporting
- Multi-browser & mobile testing
- Docker + GitHub Actions CI/CD with parallel sharding
- Test tagging (`@P0`, `@Smoke`, `@Regression`)
